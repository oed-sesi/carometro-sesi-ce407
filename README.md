# Carômetro Escolar — SESI 407

Sistema de consulta de alunos por foto.  
Frontend 100% estático hospedado no GitHub Pages.

## Acesso

🔗 **[Acessar Carômetro](https://SEU-USUARIO.github.io/carometro-sesi407/)**

> Restrito a professores, coordenadores e equipe escolar.

## Estrutura

```
carometro-sesi407/
├── index.html          ← Aplicação principal (HTML + CSS + JS)
├── data/
│   ├── alunos.json     ← Dados dos alunos
│   └── config.json     ← Configuração e usuários
├── images/             ← Fotos dos alunos (matricula.jpg)
├── assets/
│   └── img/
│       └── logo_sesi.png
└── scripts/            ← Utilitários de manutenção
```
